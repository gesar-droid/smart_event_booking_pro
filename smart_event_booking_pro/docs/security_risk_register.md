# Security and Risk Register
| Risk | Impact | Mitigation |
|---|---|---|
| Weak passwords | Unauthorised access | Password hashing and minimum length |
| SQL injection | Data breach | PDO prepared statements |
| Cross-site scripting | Malicious script execution | htmlspecialchars output escaping |
| Unauthorised admin access | Data misuse | Role-based access checks |
| AI incorrect description | Misleading information | Human review before saving |
