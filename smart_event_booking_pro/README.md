## How to Run in XAMPP
1. Copy `smart_event_booking_pro` folder to `C:/xampp/htdocs/`.
2. Start Apache and MySQL in XAMPP.
3. Go to `http://localhost/phpmyadmin`.
4. Import `database.sql`.
5. Open: `http://localhost/smart_event_booking_pro/login.php`.

## Features
- Secure login and registration
- Admin and standard user roles
- Event CRUD management
- Search and category filter
- Ticket booking system
- Booking records
- Audit logs
- AI-style event description helper with human review
- Responsive professional dashboard UI

## Responsible AI Use Statement
The AI helper in this project is a local rule-based assistant that creates draft event descriptions. It does not send sensitive data to external AI tools. Admin users must review and edit all AI-suggested content before saving or publishing.
